# Causal Tools for Sports Analytics: Evaluating Defensive Strategies in MLB

## Overview
This workshop introduces core ideas in causal inference through an applied sports analytics example. Using the MLB infield shift as a motivating case study, we show how observational data can be used to evaluate defensive strategies and estimate their effects on game outcomes.

## Level
Advanced

## Main Teaching Material
Primary paper: *Causal effect of the infield shift in the MLB* by Markes, Wang, Gronsbell, and Evans (2024).

## Learning Objectives
By the end of the workshop, participants will be able to:
1. Explain the difference between association and causation in sports analytics.
2. Describe confounding using the potential outcomes framework and causal diagrams.
3. Understand the intuition behind matching and weighting.
4. Interpret a causal analysis of the MLB infield shift.

## Prerequisites
- Basic familiarity with introductory probability
- Some exposure to Rstudio
